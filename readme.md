# outfield

count strikes away from **at the money** to options priced <= $0.10


### install

`npm install`

to [spit](https://github.com/seandunaway/spit) at open

`crontab ./crontab`


### run

`./index.mjs`

optionally pass magic number as first argument


### output

`+6 -9`

6 call strikes above, 9 put strikes below


### uninstall

`crontab -r`
